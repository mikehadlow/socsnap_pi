socsnap_pi
==========

Twitter based social picture booth for Raspberry Pi.

Install ...

    sudo apt-get install liboauth-dev
    sudo apt-get install libcurl-dev
    
    wget http://download.zeromq.org/zeromq-3.2.3.tar.gz
    tar -zxvf zeromq-3.2.3.tar.gz
    cd zeromq-3.2.3
    ./configure
    make
    make install
